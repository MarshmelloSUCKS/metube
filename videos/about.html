About

This GitLab repository is an open-source version of MeTube, an idea I had one day that I decided to make. MeTube (not to be confused with YouTube) is a place where NeoCities users can host content they've made. All you need to do is download the repository, place the "videos" folder in your root directory, edit the pages, and share! MeTube uses HTML5 videos, so browsers made with support for H.264 decoding and HTML5 video will be able to play whatever MP4 or WebM videos you host.

This site is in no way affiliated with YouTube.
